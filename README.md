#### Prerequisites

- [NodeJS v14+](https://nodejs.org)

#### Testing Locally

- Create a `.env` file at the root of the project.


#### Adding Sound Effects

> Caveats:
> - Only supports MP3s at this time.

- Find a sound effect you'd like to add.
- Upload it to the `effects` folder.
